Project 01 Detection SIEM Featured

Centralized SIEM & Intrusion Detection System

Engineered a unified threat detection pipeline by deploying a Wazuh SIEM Manager integrated with a Suricata Network Intrusion Detection System, aggregating host and network telemetry into a single pane of glass.

How It Works

Deployed Wazuh SIEM Manager as the central log aggregation and alerting engine using Docker Compose

Integrated Suricata NIDS for real-time network packet analysis and signature-based threat detection

Installed HIDS agents across multiple OS architectures (Ubuntu amd64, Raspberry Pi aarch64) for cross-platform endpoint visibility

Correlated authentication failures (auth.log) and network events (eve.json) to map active threats against the MITRE ATT&CK framework

Tools & Technologies

Wazuh Suricata Docker Compose Ubuntu Server MITRE ATT&CK JSON Log Parsing

Key Outcomes

Achieved full network and endpoint visibility by correlating NIDS and HIDS telemetry in a unified SIEM dashboard

Successfully mapped detected threats to MITRE ATT&CK techniques for structured incident analysis

Eliminated blind spots across heterogeneous operating systems and hardware architectures

Project 02 Defense Perimeter

Encrypted VPN & DNS Threat Sinkhole

Engineered a secure remote access gateway using WireGuard VPN with cryptokey routing, paired with a network-wide Pi-hole DNS sinkhole to block malicious domains and detect C2 beaconing at the network edge.

How It Works

Configured WireGuard VPN server with cryptokey routing and custom split-tunneling rules (AllowedIPs) for secure, encrypted remote network access

Modified edge router DHCP settings to force all network devices through a locally hosted Pi-hole DNS sinkhole

Integrated Pi-hole DNS query logs directly into the SIEM using a custom-compiled ARM agent for centralized threat visibility

Deployed on a Raspberry Pi running Linux/ARM, managing NAT traversal, UDP port forwarding, and UFW firewall rules

Tools & Technologies

WireGuard Pi-hole Raspberry Pi Linux/ARM UFW Firebog Blocklists DNS

Key Outcomes

Eliminated insecure remote access by establishing encrypted VPN tunnels accessible from any location worldwide

Blocked telemetry and malicious domains network-wide, significantly reducing outbound threat surface

Enabled real-time detection of potential C2 beaconing through DNS log correlation in the SIEM

Project 03 Offense Validation

Threat Emulation & Incident Validation Environment

Built a controlled adversary simulation lab using Kali Linux and Metasploitable to execute real attack chains — from reconnaissance to credential compromise to authenticated access — and validate that the SOC stack detects every stage.

How It Works

Provisioned a Metasploitable Docker container as a deliberately vulnerable target on the internal network

Deployed a dedicated Kali Linux VM as the internal adversary platform via VirtualBox

Executed Nmap SYN stealth scans with OS detection (-sS -O) and SSH brute-force attacks using Hydra to successfully compromise target credentials

Performed post-exploitation by authenticating with compromised credentials, completing the full attack chain from reconnaissance to access

Validated that NIDS and SIEM alerting thresholds accurately captured and categorized all IoCs at every stage of the attack

Tools & Technologies

Kali Linux VirtualBox Metasploitable Nmap SSH Docker

Key Outcomes

Executed a full attack chain — recon, brute force, credential compromise, authenticated access — and confirmed the SIEM captured every stage

Demonstrated the "Red Team mindset" essential for effective Blue Team defense

Identified and resolved gaps in alert thresholds before they became blind spots
